Mo.API

Mo.js 内部扩展函数，包括 UI 、交互 以及其他。

使用动态方式加载 API 请参考：Mo.script()

注意：直接引用以下脚本时，不同编码下需要 charset 来申明编码，否则可能会出现中文乱码的问题。

文件 API 描述
mo.ajax.js Mo.Ajax() Ajax 组件
mo.drag.js Mo.Drag() Drag 组件

  Mo.Message() 页面消息组件
  Mo.Tips() 小纸条/小提示
  Mo.FadeBox() 淡入淡出幻灯片
  Mo.TabMenu() 项卡/滑动门
  Mo.Soler() 标签选择器
mo.effect.js Mo.Grade() 星级评分组件
  Mo.DateDiff() 倒计时函数
  Mo.Process() 进度条
mo.form.js Mo.Form() 表单验证工具
  Mo.Config() 表单项配置工具
  Mo.Serialize() 表单序列化工具
  Mo.Password() 密码强度测试
mo.hash.js Mo.base64() 将字符串 base64 编码/解码
  Mo.sha1() 字符串 sha1 加密
  Mo.md5() 字符串 MD5 加密
mo.interface.js Mo.Command() 浏览器交互命令
  Mo.Clipboard() 复制到剪贴板
