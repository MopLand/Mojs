Mo.Validate

对象验证方法。

Mo.Validate.Array( object )

验证对象是否为数组。

object 需要验证的对象

 Mo.Validate.Array( [1,2,3] );

Mo.Validate.Function( object )

验证对象是否为函数。

object 需要验证的对象

 Mo.Validate.Function( function(){} );

Mo.Validate.Object( object )

验证对象是否为对象。

object 需要验证的对象

 Mo.Validate.Object( alert );

Mo.Validate.Date( object )

验证对象是否为日期。

object 需要验证的对象或字符串

 Mo.Validate.Date( new Date() ); 

 Mo.Validate.Date( '2012-12-22' );

Mo.Validate.Number( object )

验证对象是否为数值。

object 需要验证的对象

 Mo.Validate.Number( 9 );

Mo.Validate.String( object )

验证对象是否为字符串。

object 需要验证的对象

 Mo.Validate.String( 'Mo' );

Mo.Validate.Defined( object )

验证对象是否已定义。

object 需要验证的对象

 Mo.Validate.Defined( Mo );

Mo.Validate.Empty( object )

验证对象是否为空。

object 需要验证的对象

 Mo.Validate.Empty( '' );

Mo.Validate.Boolean( object )

验证对象是否为布尔值。

object 需要验证的对象

 Mo.Validate.Boolean( false );

Mo.Validate.Window( object )

验证对象是否为 window 。

object 需要验证的对象

 Mo.Validate.Window( window );

Mo.Validate.Document( object )

验证对象是否为 document 。

object 需要验证的对象

 Mo.Validate.Document( document );

Mo.Validate.Element( object )

验证对象是否为元素。

object 需要验证的对象

 Mo.Validate.Element( document.body );

Mo.Validate.Chinese( string[, all] )

验证字符串是否包含中文。

string 需要验证的字符串

all 可选，ture 或 false，是否全部为中文

 Mo.Validate.Chinese( '我是中文' );

验证字符串是否全部由中文构成。

 Mo.Validate.Chinese( '我是中文和 e 文' , true );

Mo.Validate.Safe( string )

验证对象是否为安全字符，即是否有特殊字符。

string 需要验证的字符串

 Mo.Validate.Safe( '[]' );

Mo.Validate.Email( string )

验证对象是否为电子邮件地址。

string 需要验证的邮件地址

 Mo.Validate.Email( 'example@domain.com' );

Mo.Validate.URL( string )

验证对象是否为 URL 资源。

string 需要验证的URL 资源

 Mo.Validate.URL( 'http://example.domain.com' );

Mo.Validate.IP( string )

验证对象是否为 IP 地址。

string 需要验证的IP地址

 Mo.Validate.IP( '127.0.0.1' );

Mo.Validate.ID( string )

验证字符串是否为身份证。

string 需要验证的身份证号码，支持15或18位

 Mo.Validate.ID( '42112119801010X' ); 

 Mo.Validate.ID( '421121198010104283' );

Mo.Validate.Password( string )

验证对象是否为合法密码（6~20 位）。

string 需要验证的密码字符串

 Mo.Validate.Password( '123456' );

Mo.Validate.Color( string )

验证对象是否为颜色。

string 需要验证的颜色字符串，16进制颜色

 Mo.Validate.Color( '#AABBCC' );

Mo.Validate.Phone( string )

校验普通电话、传真号码：可以“+”开头，除数字外，可含有“-”。

string 需要验证的号码字符串

 Mo.Validate.Phone( '6666888' ); 

 Mo.Validate.Phone( '0519-6666888' );

Mo.Validate.Mobile( string )

校验手机号码：必须以数字开头，除数字外，可含有“-”。

string 需要验证的手机号码字符串

 Mo.Validate.Mobile( '15066661234' );
