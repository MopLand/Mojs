Mo.Dialog( style, title, content, width, height, config )

执行字符串处理操作。

style 样式名称

title 对话框标题

content 主体内容

width 宽度，支持百分比

height 高宽，支持百分比

config 选项配置

config 可接受参数

locked 是否锁定窗口背景

remove 是否显示关闭按钮

draged 是否支持拖动窗口

zindex CSS z-index

unique 唯一标识，设置此参数可以防止重复打开窗口

minimize 是否显示最小化按钮

maximize 是否显示最大化按钮

create 是否立即创建对象

opacity 拖动窗口时是否透明

wrapper 窗口父容器（CSS选择器）

自定义事件方法

onRemove 窗口关闭时回调，需要 return true;

onMinimize 窗口最小化时回调，需要 return true;

onMaximize 窗口最大化时回调，需要 return true;

onCreate 窗口创建时回调

onDblclick 标题栏双击时回调

onDragStart 窗口开始移动时回调

onDragEnd 窗口结束移动时回调

Mo.Dialog( "mo-dialog", "OK", '<img src="../res/images/ok.jpg" />', 390, 0, { "remove" : true, "draged" : true, "maximize" : true, "locked" : true } );

Mo.Soler( self, event, value, style, data[, func, fX, fY] )

标签选择器。

self 当前参考对象

event event 事件

value 默认值

style 样式名称

data 数据源，JSON 形式

func 回调函数

fX 向左偏移多少，单位：像素

fY 向上偏移多少，单位：像素

func 可接受参数

value 当前选定值

self 当前参考对象，可以使用 this 获取

Mo.Soler( this, event, this.value, "soler", json, function( value ){ this.value = value; } );

Mo.Message( style, html, time[, config, func ] )

页面消息。

style 样式名称

html HTML 代码

time 显示时间，单位：秒

config 选项配置

func 回调函数

config 可接受参数

unique 唯一ID，设置此参数可以使用选择器来访问

center 是否居中于窗口

Mo.Message( "message", '我是来自页面的消息', 3, {"center":true} );

Mo.Tips( self, event, style, html, action[, fX, fY, config] )

工具提示。

self 当前参考对象

event event 事件

style 样式名称

html 默认值

action 关闭纸条的事件名称

fX 向左偏移多少，单位：像素

fY 向上偏移多少，单位：像素

config 选项配置

config 可接受参数

unique 唯一ID，设置此参数可以使用选择器来访问

bubble 是否响应 click 冒泡事件，默认为 false

Mo.Tips( this , event, 'tips', '这里是提示信息', 'mouseover', 20, 0 );

点击显示工具提示。

Mo.Tips( this , event, 'tips', '这里是提示信息', 'click', 20, 0 );
