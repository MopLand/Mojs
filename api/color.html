Mo > API > Color
Mo.Color()

常用颜色选择器。

Mo.Color( this, event, 'mo-color', function( color, obj ){ obj.style.background = color ; });

Mo.Color( this, event, 'mo-color', function( color, obj ){ obj.style.background = color ; }, 21, 0 );

Mo.Color.RGB()

将 16 进制颜色值转为 RGB 颜色值。

Mo.Color.RGB( '#FFFFFF' );

Mo.Color.HEX()

将 RGB 颜色值转为 16 进制颜色值。

Mo.Color.HEX( [255,255,255] );

Mo.Color.rand()

将产生一个随机颜色值。

Mo.Color.rand( );
