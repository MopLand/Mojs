Mo.Command( cmd , param , func )

浏览器交互命令。

cmd 需要执行的命令

param 参数值

func 回调函数

cmd 可接受参数

favorite 设置为书签（添加到收藏）

homepage 设置为主页

param 可接受参数（json 格式）

title 标题名称

href 链接地址

Mo.Command( 'favorite', {'title':'我是标题', 'href':location.href} );

加入收藏 

Mo.Command( 'homepage' );

设为首页

Mo.Clipboard( text , func )

复制到剪贴板。

text 文本内容

func 回调函数 

Mo.Clipboard( document.title+'\n'+location.href , function( text ){ alert('已复制如下信息到剪贴板:\n\n'+text); } );

复制到剪贴板
