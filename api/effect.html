Mo.DateDiff( expire[, func] )

倒计时。

expire 过期时间,Unix 时间戳（秒）

func 函数名称或匿名函数

func 可接受参数

diff -1 为已过期，否则为相差的秒数

date 返回相差的时间值，JSON 对象，{ "d":"天", "h":"小时", "m":"分", "s":"秒" }

Mo.DateDiff();
