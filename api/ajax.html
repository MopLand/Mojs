Mo > API > Ajax
Mo.Ajax("...")

异步交互请求

var ajax = new Mo.Ajax("example.html");

这里将会显示 Ajax 内容！

事件： 描述：
onLoading 开始加载时响应
onLoaded 加载完成时响应
onInteractive 开始交互时响应
onCompletion 读取完毕时响应
onError 遇到错误时响应
.setVar()

设置请求的参数和值。

ajax.setVar({
"key":"val",
"&get":"abc"
});

.setHeader()

设置请求的头部信息。

ajax.setHeader( "Content-Type" , "application/x-www-form-urlencoded" );
.method

设置发送请求的方式。

ajax.method = "GET";

以下可选值，默认为 GET：

值： 描述：
GET 完全 GET 提交
POST 完全 POST 提交
GET&POST GET 与 POST 混合提交
.getURL()

返回请求的URL地址。

ajax.getURL();

.send()

发送请求。

ajax.send();
