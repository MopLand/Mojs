Mo.Toolkit

实用工具箱，提供浏览器视口、滚动条位置等获取接口。

.is( object )

判断对象类型。

object 需要验证的对象

Mo.Toolkit.is( [1,2,3] );

Mo.Toolkit.is( document.body );

.isOnDom( node )

判断目标节点是否存在于 DOM 中。

node 节点对象

Mo.Toolkit.isOnDom( document.body );

.contains( a, b )

检测 b 节点是否包含在于 a 元素中。

a 节点对象

b 节点对象

Mo.Toolkit.contains( document.body, Mo.$('wrapper') );
.getViewportSize()

获取当前视口信息。

Mo.Toolkit.getViewportSize( );
.getClinetRect( ele )

获取元素在视口内的位置信息。

ele 元素对象

Mo.Toolkit.getClinetRect( Mo.$('element') );

.getScrollPosition( )

获取当前滚动条信息。

Mo.Toolkit.getScrollPosition( );
